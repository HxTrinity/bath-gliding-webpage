# University of Bath Gliding Webpages

This is the codespace for the University of Bath Gliding website.

## HTML Files
The HTML filenames refer to the webpage it relates to the ending URL on the SU website. `main.html` is the only exception. This is the main webpage.

The HTML files is written in a way where you can copy and paste it directly into the edit pages box after you disable WYSIWYG editor. (Uncheck the box). Therefore this HTML code is not complete to run on a seperate website.

## How to Use
1. Make sure you have completed the edit webpage survey. This gives you access to edit the webpage
2. Go to the page you want to edit, click your profile on the top right and press edit page
3. Disable "Use WYSIWYG editor".
4. Copy and paste the correct `.html` file in.

## Known Limitations, Bugs & Fixes

If you enable WYSIWYG acidentally, some CSS styling breaks. To fix this, do not enable WYSIWYG. If you do accidentally break the CSS because you switched it to it accidentally, replace the website content with the word "blah", save it and then publish it (making sure WYSIWYG is enabled). Then disable WYSIWYG, copy and paste the original `.html` code from this repo onto the website, save it and publish. This should fix the broken CSS.

Do not save and publish incredibly quickly. Sometimes the website is slow and it won't save your previous work! After you press save, give it a few seconds, then publish.
